Getting Started

Introduction

Welcome to the Steel UI Library. A refined, high-performance interface framework designed for seamless script integration.

Booting the Library

Load the core framework into your environment using the GitHub raw link.

local Library = loadstring(game:HttpGet('https://raw.githubusercontent.com/sd1246846/builtformen/refs/heads/main/STEEL'))()

Key System

The key system prevents unauthorized access. It creates a separate "SteelKeySystem" UI and pauses execution until the correct key is provided.

Library:InitKeySystem({
    Key = "STEEL-FREE", 
    Link = "https://discord.gg/steel-community"
})

Window Management

Create a new window instance and add tabs for categorized navigation.

local Window = Library:CreateWindow("STEEL HUB")

-- Icons use Roblox Image IDs
local MainTab = Window:AddTab("Main", "13837882830")

Interactions

All interactive elements must be placed inside a Section.

local Section = MainTab:AddSection("Combat Features")

-- Simple Button
Section:AddButton("Kill All", function()
    print("Action Triggered")
end)

-- Toggle Switch
Section:AddToggle("Auto Aim", false, function(bool)
    print("State: ", bool)
end)
